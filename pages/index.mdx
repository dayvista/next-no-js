## A static site, with an opt-in 2kb runtime + granular SSR hydration

### This page has ZERO JavaScript - as it didn't need any

But in the examples below, JS can be loaded on-demand to either hydrate the components, or just to
access the markup.

### Examples:

-   [on-demand SSR hydration](/examples/with-js)
-   [on-demand SSR hydration with `getStaticProps`](/examples/with-data)
-   [on-demand vanilla JS](/examples/vanilla-js)

-   [Examples with decentralised modules (few extra KB, but easier to scale)](https://next-no-js-git-decentralized.shaneosbourne8.vercel.app)
